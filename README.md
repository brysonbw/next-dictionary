## Usage

```bash
git clone git@github.com:brysonbw/next-dictionary.git
```

```bash
cd next-dictionary
```

```bash
rm -rf .git
```

### Install

```bash
pnpm install
```

### Developing

```bash
pnpm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build
```bash
pnpm run build
```
